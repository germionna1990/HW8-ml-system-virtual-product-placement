## Выполненные задания

Используемый стек:
- FastAPI
- Prometheus
- Docker
- Docker Compose

### 3. Проектирование ML-системы для Virtual Product Placement

Рассмотрена задача цифрового product placement в видеопотоке с использованием генеративного ИИ.

**Бизнес-задача:**  
показывать пользователям видеоролики с персонажами, похожими на целевую аудиторию, с интегрированным брендом.

Для решения выбрана **Kappa Architecture**, так как система работает как непрерывный streaming pipeline и не требует отдельного batch-слоя.

Спроектированные компоненты:
- извлечение кадров из видеопотока
- детекция человека (YOLO)
- streaming pipeline через Redis + FastStream
- audience matching
- генеративное брендирование
- compliance / safety check
- рендеринг итогового видео
- мониторинг системы

---

### 4. Обработка видео

В ноутбуке реализован демонстрационный pipeline:
- загрузка тестового видео
- детекция человека с помощью YOLO
- потоковая обработка кадров
- публикация/подписка через FastStream
- генерация обработанного видео

Используемый стек:
- Python
- OpenCV
- YOLO (Ultralytics)
- Redis
- FastStream
- NumPy
